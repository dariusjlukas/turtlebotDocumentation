To launch this version:
  1. Open a terminal where you downloaded the .tar.gz file (right-click, Open in terminal)
  2. Extract the zipped file:
    $ tar xvzf turtlebotDocumentation-linux-x64.tar.gz
  3. Enter the directory:
    $ cd turtlebotDocumentation-linux-x64/
  4. Set the file "turtlebotdocumentation" as runnable:
    $ chmod +x turtlebotdocumentation
  5. Then launch the file turtlebotdocumentaton:
    $ ./turtlebotdocumentation
