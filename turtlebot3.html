Turtlebot3 Overview
The stock turtlebot3 burger uses a raspberry pi
SBCSingle Board Computer
to run
ROSRobot Operating System
, which is what the user interacts with. There is also a embedded board called the OpenCR which runs the DYNAMIXEL-brand servo motors and houses the IMU. The OpenCR board has a variety of open ports such as UART, I2C, SPI, and CAN which the user could make use of. The OpenCR board runs off of an STM32 F7-series microcontroller, which implements and ARM Cortex-M7 processor and can be programmed using the arduino editor. The stock firmware can be found here. It should be noted that almost all components on the turtlebot3 are open source, so you can find all the software, mechanical cad, and electrical designs online; mostly on
Robotis'company in charge of turtlebot3 documentation
website.
